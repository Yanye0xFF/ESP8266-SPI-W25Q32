esp8266 GPIO模拟SPI输入/输出配置  
读取(不限字节数)
按页写(256字节)
写入(自动换页,未完成)
扇区擦除(4K)
块擦除(32K)
块擦除(64K)
整片擦除(w25q32整片擦除非常耗时,大致为40s左右)
读设备ID
读64位识别码
读JedecID
